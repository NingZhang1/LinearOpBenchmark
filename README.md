# Benchmark on Linear Algebra Operations 

这个仓库用于测试不同实现线性代数操作的性能，不同的实现方案包括

1. 手写 C++ 代码 (plain C++)
2. MKL
3. Eigen
4. ISPC
5. 模板循环展开
